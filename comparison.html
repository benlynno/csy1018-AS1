Web Developer

In return, they will offer you this career commitment - the opportunity to develop and progress in a fun and rewarding environment, where success is celebrated.

  • Exceptional JavaScript / HTML / CSS ability
  • Developing cross-browser compatible solutions
  • Experience of JavaScript frameworks
  • Experience of modern development techniques including modular CSS and component-orientated architecture
  • Ability to communicate your UX decisions with your peers and less technical colleagues

Game Developer

Location: London

Job Description:Game Developer / Software Engineer (Java REST JavaScript Unity NoSQL SQL AWS CI/CD).
As a Game Developer you will be designing code architecture, implementing, debugging and optimising systems to extend the game feature set; ensuring quality code and supporting others with code and design reviews as part of Agile team.
Requirements:
  • Strong software development experience including game development, Java experience is required, Unity is highly desirable and you will be using this going forward on the frontend.
  • Good knowledge of databases SQL and NoSQL
  • Familiar with unit testing and CI/CD
  • Creative, innovative and conscientious in solving complex technical problems.
  • Flexible and comfortable working in a fast changing environment
  • Desirable: Python, Scala, Cloud technologies (AWS, Azure, Google Cloud)
Salary: (Up to £55k) plus benefits including private health care, life assurance, pension, season ticket loan and profit share scheme.

https://www.cwjobs.co.uk/
