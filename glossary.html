GLOSSARY

HTML


<link>:
element specifies relationships between the current document and an external resource

href:
mandatory for links

title:
for ‘tool tips’

src:
mandatory for images and embedded media

alt:
for images

width and height:
for images and embedded media

class:
empowering CSS and JavaScript

<strong> :
for strong emphasis (typically styled as bold)

<ol>:
for ordered lists of one or more (typically numbered) items

<ul> :
:for unordered lists of one or more (typically bullet pointed) items

<li>:
for each list item (within either <ol> or <ul>)

heading :
Heading elements like <h1>, <h2>, <h3>, ..... allowyou to use six levels of document headings,ranging from largest to the smallest.

tables :
An element for displaying information in rows and columns.

span:
Span tag encloses text for styling purposes only.

<td>:
Table Data tag encloses text data that is to be displayed in a regular table data cell.

<script>:
Script tag encloses script code, or specifies an external script resource.

<p> :
Paragraph tag encloses text and automatically adds space before and after itself to create a paragraph block.

<tbody> :
Table Body tag encloses table row and data elements,to define a table body.

<map> :
Map tag encloses a number of area elements to define an image map.

<section> :
Section tag encloses text that is a sectiopn of a document,like chapter.

<video> :
Video tag encloses source elements and spcifies video media content.

<tfoot> :
Encloses table row elements to define a table footer section.

<thead> :
Table header tag encloses table row elements to define a table hearder section.


fluid class:
Provides a full width container, spanning the entire width of the viewport.



CSS



    flexbox :
    Makes it easier to design flexible responsive layout structure without having to use floats or positioning.
    active :
    An activated element

    hover :
    An element when the cursor rolls over it

    first-child :
    An elementthat is the first sibling

    last-child :
    An element that is the final sibling

    disabled :
    An element that is enabled

    margin :
    The margin is the space around the element. The larger the margin, the more space between our element and the elements around it. Example; margin-top, margin-right,margin-left and margin-bottom.

    font-family :
    The font-family property sets the font of an HTML element's text.

    padding :
    Is the spacing between the content and the border (edge of the element). Example; padding-top,padding-right,padding-bottom and padding-left

    id: empowering in-page links, CSS and JavaScript


    Comments in CSS are signified by a forward-slash and asterisk. Example: /* This is a single line comment */

    selector:
    The way you declare which elements the styles should apply to

    float:
    It is a property used for positioning and layout on web pages.

    text-align:
    It is a property used to set the horizontal alignment of a text.

    opacity:
    It specifies the transparent of an element

    media query:
    These are a popular techniques for delivering a tailored style sheet to tablets, iPhone, and Androids.

    text-decoration:
    Its property specifies the decoration added to text.

    text-justify:
    This property specifies how justified text should be aligned and spaced.

    overflow:
    This property specifies what happens if content overflows an element's box.

    line-heights:
    This property specifies the line height.

    flex-direction:
    It specifies the direction of the flexible items.

    list-style-type:
    It specifies the type of list-item marker in a list.

    min-width:
    This prevents the value of the width property from becoming smaller than min-width.

    flex-direction:
    This property specifies the direction of the flexible items.

    text-transform:
    It controls the capitalization of text.
